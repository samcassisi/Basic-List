# Basic List

An open-source, minimal to-do list app for iOS built with SwiftUI.

## Features

- **Multiple lists** - Create, rename, reorder, and delete lists. Switch between them with swipe navigation.
- **Quick entry** - Add items from a persistent input bar. Insert new items inline while editing.
- **Drag-to-reorder** - Long-press the handle to reorder items with auto-scroll support.
- **Auto-archive** - Completed items archive automatically after a short delay. Unarchive them at any time.
- **Home Screen widgets** - Small, medium, large, and Lock Screen widgets. Toggle items complete directly from the widget.
- **Siri & Shortcuts** - Add items by voice or from the Shortcuts app.
- **Import & export** - CSV import/export and copy-all-to-clipboard.
- **Deep links** - Open specific lists or items via `basiclist://` URLs.

## Requirements

- iOS 26+
- Xcode 26+

## Getting Started

1. Clone the repository.
2. Open `Basic List.xcodeproj` in Xcode.
3. Build and run on a simulator or device.
